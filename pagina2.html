Sintaxe Básica do CSS

Uma regra CSS é composta por um seletor e um bloco de declaração:

/* Isso é um comentário CSS */
seletor {
  propriedade: valor; /* Declaração */
}
Elemento de Exemplo

Seletores CSS

Seletores são padrões usados para selecionar os elementos que você quer estilizar.

/* Seletor de elemento */
p { color: blue; }

/* Seletor de classe */
.minha-classe { font-size: 18px; }

/* Seletor de ID */
#meu-id { background: #f0f0f0; }

/* Seletor universal */
* { margin: 0; }

Box Model

O Box Model é um conceito fundamental do CSS que define como os elementos são renderizados.

.caixa {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #3498db;
  margin: 30px;
}

Cores e Fundos no CSS

O CSS oferece várias formas de definir cores e fundos para elementos.

/* Diferentes formas de definir cores */
.elemento {
  color: red; /* Nome da cor */
  background-color: #ff0000; /* Hexadecimal */
  border-color: rgb(255, 0, 0); /* RGB */
  outline-color: rgba(255, 0, 0, 0.5); /* RGBA com transparência */
  box-shadow: 0 0 10px hsl(0, 100%, 50%); /* HSL */
}
