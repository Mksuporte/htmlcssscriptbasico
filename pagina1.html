Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar a estrutura das páginas web. É a base de todo site na internet.

Diferente de linguagens de programação, o HTML é uma linguagem de marcação que define a estrutura e o significado do conteúdo.

Por que não decorar?

No HTML, o importante é entender a função de cada elemento, não decorar. Com o tempo e prática, você naturalmente memorizará as tags mais importantes.

Boas Práticas

  • Use HTML semântico sempre que possível
  • Mantenha uma estrutura hierárquica clara
  • Escreva código limpo e bem indentado
  • Valide seu HTML regularmente

Exemplo Básico

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <!-- Configurações e metadados -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Página</title>
</head>
<body>
    <!-- Conteúdo visível -->
    <header>
        <h1>Bem-vindo à Minha Página</h1>
    </header>
    
    <main>
        <p>Este é um parágrafo de exemplo.</p>
        <img src="imagem.jpg" alt="Descrição">
    </main>
    
    <footer>
        <p>Rodapé da página</p>
    </footer>
</body>
</html>

Estrutura Básica do HTML

Todo documento HTML possui uma estrutura básica composta por elementos essenciais:

  • <!DOCTYPE html> - Define que é um documento HTML5
  • <html> - Elemento raiz do documento
  • <head> - Contém metadados sobre a página
  • <body> - Contém o conteúdo visível da página

Metadados no HEAD

O elemento <head> contém informações sobre a página que não são exibidas diretamente no navegador, mas são essenciais para:

  • SEO (Otimização para Mecanismos de Busca)
  • Configuração de viewport para responsividade
  • Links para folhas de estilo CSS
  • Scripts JavaScript

Estrutura Detalhada

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <!-- Metadados -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Descrição da minha página">
    
    <!-- Links para recursos externos -->
    <link rel="stylesheet" href="estilo.css">
    
    <!-- Título da página -->
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo da página -->
    <h1>Meu Título</h1>
    <p>Meu parágrafo de texto.</p>
    
    <!-- Scripts JavaScript -->
    <script src="script.js"></script>
</body>
</html>

Elementos HTML

Os elementos HTML são os blocos de construção de qualquer página web. Cada elemento é representado por uma tag e pode ter atributos que modificam seu comportamento.

Tipos de Elementos

  • Cabeçalhos: h1 a h6
  • Texto: p, span, strong, em
  • Listas: ul, ol, li
  • Links: a
  • Imagens: img
  • Tabelas: table, tr, td
  • Formulários: form, input, button

Exemplos de Elementos

<!-- Cabeçalhos -->
<h1>Título Principal</h1>
<h2>Subtítulo</h2>

<!-- Parágrafo -->
<p>Este é um parágrafo com <strong>texto em negrito</strong> e <em>texto em itálico</em>.</p>

<!-- Lista -->
<ul>
    <li>Item da lista 1</li>
    <li>Item da lista 2</li>
</ul>

<!-- Link -->
<a href="https://exemplo.com">Visite o site</a>

<!-- Imagem -->
<img src="imagem.jpg" alt="Descrição">

<!-- Formulário -->
<form>
    <input type="text" placeholder="Digite seu nome">
    <button type="submit">Enviar</button>
</form>
