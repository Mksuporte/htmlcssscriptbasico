Aula de HTML - Estrutura Básica

Entenda a função de cada elemento sem precisar decorar!

Por que não decorar?

No HTML, o importante é entender a função de cada elemento, não decorar. Com o tempo e prática, você naturalmente memorizará as tags mais importantes.

Estrutura Básica do HTML

Todo documento HTML possui uma estrutura básica composta por:

  • <!DOCTYPE html> - Define que é um documento HTML5
  • <html> - Elemento raiz do documento
  • <head> - Contém metadados sobre a página
  • <body> - Contém o conteúdo visível da página

Dica de Aprendizado

Foque em entender a semântica (significado) de cada tag. Por exemplo:

  • Tags de cabeçalho: h1 a h6
  • Tags de texto: p, span, strong, em
  • Tags de mídia: img, audio, video
  • Tags de estrutura: div, section, article, header, footer

Exemplo de Código HTML

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <!-- Configurações e metadados -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Página</title>
</head>
<body>
    <!-- Conteúdo visível -->
    <header>
        <h1>Bem-vindo à Minha Página</h1>
    </header>
    
    <main>
        <p>Este é um parágrafo de exemplo.</p>
        <img src="imagem.jpg" alt="Descrição">
    </main>
    
    <footer>
        <p>Rodapé da página</p>
    </footer>
</body>
</html>
