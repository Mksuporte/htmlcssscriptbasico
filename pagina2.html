O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Enquanto o HTML cuida da estrutura e conteúdo, o CSS é responsável pela aparência visual.

Vantagens do CSS

O CSS oferece diversas vantagens para o desenvolvimento web:

Principais vantagens
  • Separação de preocupações: Conteúdo (HTML) e apresentação (CSS) são separados
  • Consistência visual: Mantém uma aparência uniforme em todo o site
  • Manutenibilidade: Mudanças de estilo podem ser feitas globalmente
  • Eficiência: Reduz a quantidade de código HTML necessário
  • Flexibilidade: Permite diferentes layouts para diferentes dispositivos

Como usar CSS

Existem três formas principais de aplicar CSS a documentos HTML:

1. CSS Interno
<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: lightblue;
    }
    h1 {
      color: white;
      text-align: center;
    }
  </style>
</head>
<body>
<h1>Meu Primeiro CSS</h1>
</body>
</html>
2. CSS Externo (Recomendado)
/* styles.css */
body {
  background-color: lightblue;
}

h1 {
  color: white;
  text-align: center;
}
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
<h1>Meu Primeiro CSS</h1>
</body>
</html>
3. CSS Inline
<!DOCTYPE html>
<html>
<body>
  <h1 style="color:blue;text-align:center;">
    Meu Primeiro CSS
  </h1>
  <p style="color:red;">
    Este é um parágrafo vermelho.
  </p>
</body>
</html>

Seletores CSS

Os seletores são padrões usados para selecionar os elementos que você quer estilizar.

Tipos de Seletores
/* Seletor de elemento */
p { color: blue; }

/* Seletor de classe */
.destaque { background: yellow; }

/* Seletor de ID */
#menu { width: 100%; }

/* Seletor universal */
* { margin: 0; }

/* Seletor de atributo */
a[target="_blank"] { color: red; }

Box Model

O Box Model é um conceito fundamental do CSS que define como os elementos são renderizados.

Partes do Box Model
.caixa {
  /* Conteúdo */
  width: 200px;
  height: 100px;

  /* Padding */
  padding: 20px;

  /* Border */
  border: 5px solid #3498db;

  /* Margin */
  margin: 30px;
}

Cores e Fundos

O CSS oferece várias formas de definir cores e fundos para elementos.

Formas de definir cores
.exemplo {
  /* Nome da cor */
  color: red;

  /* Hexadecimal */
  background-color: #ff0000;

  /* RGB */
  border-color: rgb(255, 0, 0);

  /* RGBA com transparência */
  outline-color: rgba(255, 0, 0, 0.5);

  /* HSL */
  box-shadow: 0 0 10px hsl(0, 100%, 50%);
}

Propriedades de Texto

CSS oferece diversas propriedades para estilizar texto.

Propriedades comuns de texto
p {
  color: #333; /* Cor do texto */
  font-family: Arial, sans-serif; /* Fonte */
  font-size: 16px; /* Tamanho */
  font-weight: bold; /* Peso: normal, bold, etc. */
  text-align: center; /* Alinhamento */
  line-height: 1.5; /* Altura da linha */
  text-decoration: underline; /* Decoração */
  text-transform: uppercase; /* Transformação */
}
