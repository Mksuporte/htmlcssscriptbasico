Comandos de Saída e Entrada

Alertas

O alert() exibe uma mensagem modal de alerta ao usuário.

// Exibe uma caixa de alerta
alert("Isso é um alerta!");

// Alert com variável
let mensagem = "Olá, mundo!";
alert(mensagem);

Confirmações

O confirm() exibe uma caixa de diálogo modal com uma mensagem e dois botões (OK e Cancelar).

// Exibe uma caixa de confirmação
let resultado = confirm("Deseja realmente excluir?");

// Verifica a resposta do usuário
if (resultado) {
    console.log("Usuário confirmou");
} else {
    console.log("Usuário cancelou");
}

Prompt

O prompt() exibe uma caixa de diálogo que solicita uma entrada do usuário.

// Exibe um prompt para entrada de dados
let nome = prompt("Qual é o seu nome?", "Digite aqui");

// Exibe o resultado
if (nome !== null) {
    alert("Olá, " + nome + "!");
}

Console e Debugging

Console

O console é uma ferramenta essencial para debugging e log de informações.

// Diferentes métodos do console
console.log("Mensagem simples");
console.info("Mensagem informativa");
console.warn("Mensagem de aviso");
console.error("Mensagem de erro");

// Exibindo variáveis
let x = 10, y = 20;
console.log("Valor de x:", x);

Console Table

O console.table() exibe dados tabulares como uma tabela.

// Array de objetos
let pessoas = [
    { nome: "João", idade: 30 },
    { nome: "Maria", idade: 25 },
    { nome: "Pedro", idade: 35 }
];

// Exibindo como tabela
console.table(pessoas);

Manipulação do DOM

Seleção e Manipulação de Elementos

O JavaScript permite selecionar e modificar elementos HTML.

// Selecionando elementos
let elemento = document.getElementById("meuId");
let elementos = document.querySelectorAll(".minhaClasse");

// Alterando conteúdo
elemento.innerHTML = "Novo conteúdo";
elemento.style.color = "red";

// Manipulando classes
elemento.classList.add("destaque");
elemento.classList.remove("antigo");

Timing Events

setTimeout e setInterval

JavaScript permite executar código após um delay ou em intervalos regulares.

// Executa após um delay
setTimeout(function() {
    console.log("Isso executa após 2 segundos");
}, 2000);

// Executa repetidamente a cada intervalo
let intervalo = setInterval(function() {
    console.log("Executando a cada 1 segundo");
}, 1000);

// Parar a execução do intervalo
clearInterval(intervalo);
