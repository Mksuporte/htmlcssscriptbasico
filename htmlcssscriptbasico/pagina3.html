Funções

Funções são blocos de código projetados para executar uma tarefa específica.

// Declaração de função
function saudacao(nome) {
    return "Olá, " + nome + "!";
}

// Chamada da função

Estruturas de Controle

Permitem controlar o fluxo de execução do código com condições e loops.

// Condicional if-else
if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

// Loop for
for (let i = 0; i < 5; i++) {
    console.log("Número: " + i);
}

Manipulação do DOM

O DOM (Document Object Model) permite que o JavaScript altere a estrutura e o conteúdo de uma página web.

// Alterar conteúdo HTML
document.getElementById("demo").innerHTML = "Novo conteúdo!";

// Alterar estilo
document.getElementById("demo").style.color = "red";

// Responder a eventos
document.getElementById("btn").addEventListener("click", minhaFuncao);

Arrays e Objetos

Arrays

Arrays são usados para armazenar múltiplos valores em uma única variável.

// Criando um array
let frutas = ["Maçã", "Banana", "Laranja"];

// Acessando elementos
console.log(frutas[0]); // "Maçã"

// Métodos de array
frutas.push("Uva"); // Adiciona no final
frutas.pop(); // Remove do final

Objetos

Objetos armazenam coleções de pares chave-valor.

// Criando um objeto
let pessoa = {
    nome: "Carlos",
    idade: 25,
    cidade: "Porto"
};

// Acessando propriedades
console.log(pessoa.nome); // "Carlos"

// Alterando propriedades
pessoa.idade = 26;
